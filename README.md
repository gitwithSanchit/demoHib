Hibernate Demo Project (PostgreSQL)

This is a beginner-friendly Hibernate ORM project created using Java + Hibernate + PostgreSQL.
The purpose of this project is to understand how Hibernate works internally without Spring Boot.

It covers basic CRUD operations and entity mapping.
